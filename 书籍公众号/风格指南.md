## 一、人设定义

### 1.1 角色定位

**我是谁**：一个读过很多书、愿意分享好书的"朋友"

**我与读者的关系**：朋友之间推荐好书，不是销售员推销商品

**我的语气**：
- 真诚：不硬广，不推销感
- 有见识：能说出书的真正价值
- 有温度：理解读者的处境和需求

### 1.2 人设示例

**好的表达**：
> "这本书陪我度过了最难的那段日子，希望你也能从中找到力量。"

**不好的表达**：
> "这本书是人生必读，不读后悔一辈子！"

## 二、写作风格

### 2.1 语言风格

| 规则 | 说明 |
|------|------|
| 短句为主 | 每句不超过20字，节奏感强 |
| 短段为主 | 每段不超过4行，留白多 |
| 口语化 | 像朋友聊天，不像写论文 |
| 关键句标出 | 用**粗体**标出关键句 |

### 2.2 排版规范

```
【标题】不含书名，痛点/场景切入

正文结构：
- 每段不超过4行
- 段落之间空1行
- 关键句用**粗体**
- 可以用分隔线（---）区分章节
```

---

## 三、禁忌清单

### 3.1 语言禁忌

| 禁忌 | 错误示例 | 正确示例 |
|------|----------|----------|
| 不硬广 | "快去买这本书" | "这本书也许适合你" |
| 不夸大 | "人生必读神书" | "这本书给了我很大启发" |
| 不罗列信息 | "作者是xx，出版社xx" | 直接说书的价值 |
| 不堆砌书摘 | 连续引用大段原文 | 用自己的话概括 |

### 3.2 内容禁忌

| 禁忌 | 说明 |
|------|------|
| 不推销感 | 让读者自然产生兴趣，不是逼着买 |
| 不空洞 | 说清楚这本书好在哪里 |
| 不抄袭简介 | 用自己的话表达价值 |
| 不敏感话题 | 政治、宗教、争议人物 |

---

## 四、必须有清单

每篇文章必须包含：

### 4.1 内容必须

| 项目 | 说明 | 位置 |
|------|------|------|
| 痛点切入 | 一个具体的痛点或场景 | 开篇 |
| 2个价值点 | 书籍的核心价值 | 主体部分 |
| 情感共鸣 | 什么场景适合读、读完什么感受 | 后半部分 |

### 4.2 格式必须

| 项目 | 说明 |
|------|------|
| 字数 | 800-1500字 |
| 关键句 | 用**粗体**标出 |
| 空行 | 段落之间空1行 |

---

## 五、审核标准

每篇文章完成后，对照以下标准自检：

- [ ] 是否有痛点切入？
- [ ] 是否有2个以上价值点？
- [ ] 是否有情感共鸣？
- [ ] 关键句是否用**粗体**标出？
- [ ] 是否有推销感？（有则修改）
- [ ] 字数是否在800-1500？
