# 书籍营销文章项目

## 目标
100 篇公众号文章，目标：卖书

## 受众
- 上班族
- 30-45 岁
- 男女均有

## 书籍方向
成长 / 职场 / 健康 / 情感

## 写作风格
- 人设：过来人/朋友视角
- 字数：1500-2000 字
- 结构：故事开场 → 痛点共鸣 → 方法/观点 → 行动建议

## 内容矩阵

| 板块 | 篇数 | 选题方向 |
|------|------|----------|
| 职场 | 30 篇 | 晋升、跳槽、向上管理、职场人际、35岁危机 |
| 成长 | 25 篇 | 认知升级、时间管理、自律、学习力、选择 |
| 健康 | 20 篇 | 久坐病、睡眠、饮食、心理压力、体检 |
| 情感 | 25 篇 | 亲密关系、亲子、父母、友情、孤独感 |

## 文章类型配比
- 痛点文（戳）：戳中读者痛点，引发共鸣
- 方法文（解）：提供可执行的方法论
- 故事文（共鸣）：真实故事，建立连接
- 观点文（立人设）：表达立场，建立信任
