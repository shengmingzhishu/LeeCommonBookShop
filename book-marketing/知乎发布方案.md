# 知乎发布方案

> 发布文章到知乎专栏/回答

---

## 一、知乎的特点

| 特点 | 说明 |
|------|------|
| 高知用户 | 用户质量高，付费意愿强 |
| 长尾流量 | 文章持续被搜索到 |
| SEO强 | 百度/Google收录好 |
| 变现方式 | 付费咨询、好物推荐、盐选专栏 |

**适合内容**：职场、成长、情感类文章

## 三、浏览器自动化方案

### 3.1 登录流程

```
1. 打开知乎登录页
2. 扫码/密码登录
3. 保存cookies到本地
4. 后续发布使用保存的登录态
```

### 3.2 发布流程

```
1. 进入专栏管理页 (zhuanlan.zhihu.com/write)
2. 点击新建文章
3. 填写标题
4. 填写内容（支持Markdown）
5. 添加话题标签
6. 设置封面图
7. 发布
```

## 四、知乎内容适配

### 4.1 标题优化

知乎标题风格：
- 提问式：`30岁后，你的腰还好吗？`
- 观点式：`我认为自律的本质是找到节奏`
- 经验式：`独居这些年，我学会的5件事`

### 4.2 内容适配

| 原格式 | 知乎格式 | 说明 |
|--------|----------|------|
| Markdown | Markdown | 直接支持 |
| 表格 | 表格 | 支持 |
| 代码块 | 代码块 | 支持 |
| 图片 | 需上传 | 先上传到知乎图床 |

### 4.3 话题选择

职场类文章推荐话题：
- `职场` `职业发展` `跳槽` `升职加薪`
- `35岁` `中年危机` `职场焦虑`

情感类文章推荐话题：
- `情感` `亲密关系` `婚姻` `孤独`
- `独居` `单身` `生活`

## 五、发布策略

### 5.1 文章 vs 回答

| 类型 | 流量来源 | 适合场景 |
|------|----------|----------|
| 专栏文章 | 关注者 + 搜索 | 长篇原创内容 |
| 回答问题 | 问题流量 | 热门问题下引流 |

**策略**：
1. 先发布专栏文章
2. 搜索相关问题
3. 用文章内容回答，附上专栏链接

### 5.2 发布时间

知乎用户活跃时间：
- 工作日：9-10点、12-13点、21-23点
- 周末：10-12点、20-22点
