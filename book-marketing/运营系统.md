---

## 一、账号矩阵

## 五、运营流程

### 5.1 日常流程（每天）

```
08:00 - 检查昨日数据，更新 analytics/
10:00 - 检查今日发布排期
14:00 - 生产下一批内容（如需要）
20:00 - 确认发布状态
21:00 - 发布（如排期）
```

### 5.2 周流程（每周）

```
周一 - 上周数据复盘
周二 - 调整本周计划
周三 - 素材库补充
周四 - 选题优化
周五 - 批量生产
周六 - 质量检查
周日 - 下周排期
```

### 5.3 月流程（每月）

```
月初 - 月度目标设定
月中 - 中期检查
月末 - 月度复盘 + 收益统计
```

## 六、关键指标

### 6.1 内容指标

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 发布频率 | 2-3篇/周/账号 | 保持稳定 |
| 阅读率 | >10% | 打开率 |
| 完读率 | >30% | 滑到末尾比例 |
| 分享率 | >3% | 转发/阅读 |
| 评论率 | >1% | 互动强度 |

### 6.2 变现指标

| 指标 | 目标 | 说明 |
|------|------|------|
| 转化率 | >2% | 阅读→购买 |
| 客单价 | 30-50元 | 单次购买金额 |
| 复购率 | >20% | 再次购买比例 |
| 月收益 | 待定 | 根据粉丝量调整 |

### 6.3 账号指标

| 指标 | A号（情感） | B号（职场） |
|------|-------------|-------------|
| 粉丝增长 | 关注 | 关注 |
| 互动率 | 关注 | 关注 |
| 取关率 | <1%/周 | <1%/周 |

## 七、Levis 养成日志同步

**同步频率**：每半天一次

**同步内容**：
- 当前进度
- 最新数据
- 关键事件
- 下一步计划

**触发条件**：
- 心跳检查时自动同步
- 重要事件发生时立即同步
- 用户要求时手动同步

## 八、风险控制

| 风险 | 应对 |
|------|------|
| 账号被封 | 内容合规审查，敏感词过滤 |
| 流量下滑 | 数据监控，及时调整选题 |
| 变现效果差 | A/B测试，优化带货文案 |
| 内容枯竭 | 持续补充素材库 |
| 发布延误 | 提前准备，建立库存 |
