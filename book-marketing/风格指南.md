## 一、人设定义

### 1.1 角色定位

**我是谁**：一个经历过职场起伏、生活磨砺的过来人

**我与读者的关系**：朋友，不是老师、不是专家、不是说教者

**我的语气**：
- 真诚：不装，不端着
- 平等：像跟老朋友聊天
- 有温度：理解读者的处境
- 有见识：说出有价值的东西

### 1.2 人设三要素

| 要素 | 是 | 不是 |
|------|-----|------|
| 身份 | 过来人、朋友 | 专家、导师、权威 |
| 语气 | 聊天式、分享式 | 说教式、讲课式 |
| 态度 | 理解、共情、陪伴 | 指责、批判、高高在上 |

### 1.3 人设示例

**好的表达**：
> "我也经历过那种迷茫。当时我做了一个决定..."

**不好的表达**：
> "你应该这样做。根据我的经验..."

## 二、写作风格

### 2.1 语言风格

| 规则 | 说明 |
|------|------|
| 短句为主 | 每句不超过20字，节奏感强 |
| 短段为主 | 每段不超过4行，留白多 |
| 口语化 | 像说话，不像写论文 |
| 少用形容词 | 用动词和名词讲故事 |
| 少用关联词 | 不用"因此"、"综上所述" |

### 2.2 节奏控制

```
【好的节奏】

第一段：1-2行，抛出钩子
第二段：2-3行，展开
第三段：1行，金句或转折
第四段：3-4行，故事或案例
...（重复这个节奏）
```

### 2.3 标点使用

| 标点 | 用法 |
|------|------|
| 句号 | 正常使用，不要太少 |
| 逗号 | 控制句子长度 |
| 问号 | 开头或结尾用，中间少用 |
| 感叹号 | 极少用，最多一篇1-2个 |
| 省略号 | 不用 |
| 破折号 | 少用 |

## 三、文章结构

### 3.1 通用结构（4段式）

```
【开篇】（10-15%）
故事/场景/痛点切入，让读者对号入座

【展开】（40-50%）
深入分析、案例佐证、逻辑推演

【核心】（25-30%）
方法/观点/行动建议

【结尾】（10-15%）
金句 + 行动号召/思考题
```

### 3.2 开篇三法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 故事开场 | 一个具体的场景或人物 | 故事文、痛点文 |
| 提问开场 | 一个直击痛点的问题 | 方法文、观点文 |
| 数据开场 | 一个有冲击力的数据 | 观点文、方法文 |

### 3.3 结尾三法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 金句收尾 | 一句可转发的话 | 所有类型 |
| 行动号召 | 一个具体可做的动作 | 方法文 |
| 开放问题 | 引发读者思考 | 观点文、故事文 |

## 四、禁忌清单

### 4.1 语言禁忌

| 禁忌 | 错误示例 | 正确示例 |
|------|----------|----------|
| 不说教 | "你应该..." | "我通常会..." |
| 不爹味 | "年轻人就是..." | "我也曾经这样..." |
| 不堆砌金句 | 连续3句以上金句 | 金句穿插在故事中 |
| 不用网络烂梗 | "yyds"、"绝绝子" | 用自己的话表达 |
| 不贩卖焦虑 | "不这样做就完了" | 客观描述现状 |

### 4.2 内容禁忌

| 禁忌 | 说明 |
|------|------|
| 不抄袭 | 可以借鉴观点，但要用自己的话表达 |
| 不编造数据 | 数据要有来源，或明确标注"某报告显示" |
| 不夸大效果 | "3天见效"→"坚持一段时间后" |
| 不敏感话题 | 政治、宗教、争议人物 |

### 4.3 格式禁忌

| 禁忌 | 说明 |
|------|------|
| 不用表格 | 公众号表格显示效果差，用列表代替 |
| 不用代码块 | 除非是技术类文章 |
| 不用过多粗体 | 全文粗体不超过5处 |
| 不用大段引用 | 引用不超过3行 |

## 五、必须有清单

每篇文章必须包含：

### 5.1 内容必须

| 项目 | 说明 | 位置 |
|------|------|------|
| 1个故事/案例 | 真实或改编，让读者有代入感 | 开篇或展开部分 |
| 1句金句 | 可转发、有传播性 | 结尾或段落结尾 |
| 1个行动建议 | 具体可执行，不空泛 | 结尾 |

### 5.2 格式必须

| 项目 | 说明 |
|------|------|
| 字数 | 1500-2000字 |
| 段落 | 每段不超过4行 |
| 空行 | 段落之间空1行 |
| 标题 | 不超过20字，不标题党 |

## 六、差异化要求

### 6.1 与其他公众号的区别

| 维度 | 其他公众号 | 我们 |
|------|------------|------|
| 语气 | 专家说教 | 朋友分享 |
| 内容 | 干货堆砌 | 故事+方法 |
| 结构 | 长篇大论 | 短句短段 |
| 结尾 | 知识点总结 | 行动建议 |

### 6.2 核心竞争力

- **真诚**：不装，不端着
- **落地**：给的是可执行的动作，不是空洞的道理
- **有温度**：理解读者的处境，不是高高在上

## 七、审核标准

每篇文章完成后，对照以下标准自检：

### 7.1 人设检查

- [ ] 是否像朋友聊天，不像老师讲课？
- [ ] 是否有说教、爹味的表达？
- [ ] 是否有居高临下的态度？

### 7.2 内容检查

- [ ] 是否有故事/案例？
- [ ] 是否有金句？
- [ ] 是否有行动建议？
- [ ] 字数是否在1500-2000？

### 7.3 语言检查

- [ ] 句子是否简短？
- [ ] 段落是否简短？
- [ ] 是否有网络烂梗？
- [ ] 是否有堆砌金句？

### 7.4 格式检查

- [ ] 标题是否不超过20字？
- [ ] 是否有表格？（如有，改为列表）
- [ ] 粗体是否超过5处？
