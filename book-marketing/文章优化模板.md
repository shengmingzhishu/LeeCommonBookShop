# 文章优化模板

> 让用户愿意停留、产生价值感

---

## 一、标题优化

**原则**：悬念 > 痛点 > 好奇 > 平铺

| 原标题 | 优化后 | 类型 |
|--------|--------|------|
| 那个同事教会我三件事 | 三年没升职，领导终于说了实话 | 悬念 |
| 为什么你做得越多... | 为什么你做得越多，领导越看不见 | 痛点 |
| 升职最快的人... | 升职最快的人，都在偷偷做这件事 | 好奇 |

---

## 二、开场优化

**原则**：冲突前置 + 制造悬念 + 情感共鸣

**优化前**：
```
小王是我见过最努力的同事...
```

**优化后**：
```
"你觉得努力就能升职？"

领导这句话，让小王彻底傻了。

他连续三年考评优秀，加班最多，从没喊过累。

但升职名单里，永远没有他。
```

---

## 三、结构优化

**原则**：表格化 + 清单化 + 模板化

**加入元素**：

1. **诊断表格**：让读者对号入座
```
| 错误 | 表现 | 后果 |
|------|------|------|
| 只做事不汇报 | 以为领导知道 | 领导不知道 |
```

2. **行动清单**：让读者可执行
```
| 序号 | 行动 | 截止 |
|:----:|------|------|
| 1 | 发周报 | 今天 |
```

3. **周报模板**：让读者可直接用
```
【本周完成】...
【下周计划】...
【需要支持】...
```

---

## 四、互动元素

**原则**：让读者参与，增加停留时间

**加入结尾**：
```
> 💡 **你是哪种"小王"？**
> 
> A. 只干活不汇报
> B. 汇报了但没亮点
> C. 有亮点但没人信
> 
> 评论区告诉我，我来帮你拆解
```

---

## 五、共情深化

**原则**：先共情，再说理

**加入段落**：
```
**你有没有过这种感觉？**

明明付出了全部，却像一拳打在棉花上。

没人看见，没人认可。

如果你也有过这种无力感，这篇文章写给你。
```

---

## 六、金句强化

**原则**：每篇至少1句可转发的金句

**示例**：
- 努力只是入场券，让努力被看见才是通行证
- 晋升不是奖赏，是投资
- 选择比努力更重要

---

## 七、优化检查清单

- [ ] 标题是否吸引人（悬念/痛点/好奇）
- [ ] 开场是否有冲突/悬念
- [ ] 是否有共情段落
- [ ] 是否有表格/清单
- [ ] 是否有可复制的模板
- [ ] 是否有金句
- [ ] 是否有互动元素
- [ ] 结尾是否有行动号召
