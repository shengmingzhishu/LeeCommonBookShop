# 被领导误解时，不要急着解释，我终于明白了

被领导误解，是职场中最让人委屈的事之一。

你明明做得很好，却被说做得不好。你明明没有那个意思，却被理解成那个意思。

第一反应，往往是解释：不是这样的，领导你听我说……

但很多情况下，越解释越糟。

因为领导在情绪上，他听不进去。你的解释，在他看来像是推卸责任、找借口。

那被误解了怎么办？

第一，先控制情绪，不要当场对抗。

被误解时，人会本能地想反驳。但这时候的反驳，往往没有好结果。

深呼吸，先稳住。可以简单说一句：我理解您的想法，让我先确认一下情况。

不要让气氛升级。

第二，等领导情绪过去，再找机会沟通。

时机很重要。等领导冷静了，或者换个场合，沟通效果会好很多。

第三，用事实说话，而不是辩解。

不要说"不是我的问题"，而是客观陈述事实：这件事的过程是这样的，我的角色是这样的，结果是那样的。

让事实为自己说话，比辩解有说服力。

第四，反思自己为什么会被误解。

有时候，误解是有原因的。可能是沟通不够清楚，可能是汇报不够及时，可能是平时的形象让对方产生了预判。

找到原因，改进自己，避免下次再被误解。

有个朋友，被领导误解后，没有急着解释。

他先冷静下来，整理了事情的来龙去脉，用邮件发给了领导。邮件里，他没有辩解，只是客观陈述了事实。

第二天，领导找他聊，说看了邮件，了解情况了。误解消除了，反而对他更信任了。

被误解，是很委屈。但委屈解决不了问题。

冷静、等时机、用事实说话、反思改进。

职场不是辩论赛，赢了道理，输了关系，得不偿失。

被误解时，先稳住，再化解。

这才是成熟的做法。
