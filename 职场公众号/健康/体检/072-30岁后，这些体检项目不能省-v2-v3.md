# 30岁后，这些体检项目不能省，我终于明白了

很多人觉得，自己年轻，体检没必要。

或者体检了，只做最基础的项目。

**但30岁之后，身体开始走下坡路。**

**有些体检项目，真的不能省。**

---

## 30岁后必检项目

| 类型 | 项目 | 频率 |
|-----|-----|-----|
| 基础项目 | 血压、血糖、血脂 | 每年 |
| | 肝功能、肾功能 | 每年 |
| | 血常规、尿常规 | 每年 |
| | 心电图 | 每年 |
| 超声检查 | 肝胆胰脾肾、甲状腺 | 每年 |
| | 乳腺超声（女性） | 每年 |
| | 妇科超声（女性） | 每年 |
| 肿瘤筛查 | CEA、AFP等肿瘤标志物 | 每年 |
| | 胃镜、肠镜（有胃病/肠道问题） | 按需 |
| 其他 | 骨密度（特别是女性） | 按需 |
| | 眼底检查 | 每年 |

---

## 体检前后要注意什么

| 时间 | 注意事项 |
|-----|---------|
| 前一天晚上 | 清淡饮食，不喝酒，10点后不进食 |
| 当天 | 空腹去医院，穿宽松衣服，带以前的体检报告 |
| 体检后 | 认真看报告，异常指标要重视，建立健康档案 |

---

## 体检投资回报

| 投入 | 回报 |
|-----|-----|
| 每年几百块体检费 | 省下未来几万块医药费 |
| 几小时时间 | 早发现早治疗，避免大问题 |
| 了解自己身体 | 有针对性地调整生活习惯 |
