# 跳槽前，先问自己三个问题

很多人跳槽，是因为"受不了了"。

领导不好、薪资太低、没有发展、同事难相处……

一气之下，就开始投简历。

**但跳槽真的能解决问题吗？**

在做出决定之前，先问自己三个问题。

---

## 第一个问题：我为什么想跳槽？

| 跳槽原因 | 是否该跳 |
|---------|---------|
| 公司经营不善、行业下滑、部门裁员 | 该跳，越早越好 |
| 和领导关系不好、工作不开心、没成长 | 先想想是不是自己的问题 |

**如果是自己的问题，换多少个地方都没用。**

---

## 第二个问题：我准备好了吗？

跳槽不是想跳就跳，需要准备：

| 准备项 | 检查问题 | 你的答案 |
|-------|---------|---------|
| 能力准备 | 你的技能在市场上有竞争力吗？ | |
| 成果准备 | 你有拿得出手的项目或业绩吗？ | |
| 经济准备 | 三个月找不到工作，你能撑住吗？ | |
| 心理准备 | 你能接受新环境的不确定性吗？ | |

**如果没有准备好，跳槽的风险会很大。**

---

## 第三个问题：新公司真的更好吗？

很多人跳槽，是因为对新公司有幻想。

但现实往往是——新公司有自己的问题，可能比你原来的还严重。

| 了解项 | 怎么了解 |
|-------|---------|
| 公司文化 | 问在那边工作的人 |
| 加班情况 | 查看职场社交平台评价 |
| 团队氛围 | 面试时多问团队情况 |
| 发展空间 | 问清楚晋升机制 |

**不要被面试时的美好印象迷惑。**

**多打听，多了解，再做决定。**

---

## 跳槽决策清单

| 序号 | 检查项 | 你的判断 |
|:----:|-------|---------|
| 1 | 跳槽原因是否合理？ | |
| 2 | 能力是否准备好？ | |
| 3 | 成果是否拿得出手？ | |
| 4 | 存款是否足够？ | |
| 5 | 新公司是否真的更好？ | |

**如果答案都是正向的：** 你可以开始准备跳槽了。

**如果有不确定的地方：** 再等等，先提升自己。

**如果只是逃避问题：** 先想想怎么解决问题。
