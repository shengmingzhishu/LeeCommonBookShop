# 为什么你学了就忘

你可能有过这样的经历——

听了一堂课，当时觉得很有收获。

但过了一周，什么都不记得了。

看了一本书，当时觉得醍醐灌顶。

但过了一个月，内容都忘了。

**为什么学了就忘？**

---

## 遗忘是正常的

首先，遗忘是正常现象。

人脑不是硬盘，不会把所有信息都存下来。

根据遗忘曲线，**学完一天后，人会遗忘70%的内容。**

这不是你的问题，是人脑的特点。

但有些方法，可以让遗忘变慢，让记忆更深。

---

## 为什么学了记不住

| 原因 | 说明 |
|------|------|
| 只有输入没有输出 | 只是听、看，没有加工 |
| 没有复习 | 学一次就想记住，不可能 |
| 没有应用 | 学了不用，知识只是信息 |
| 没有关联 | 新旧知识没有建立连接 |

---

## 📋 四种高效学习方法

| 方法 | 怎么做 | 效果 |
|------|--------|------|
| **费曼学习法** | 用自己的话讲给别人听 | 讲得清楚才是真懂 |
| **间隔重复** | 过几天复习一次，再过几天再复习 | 记忆更牢固 |
| **学以致用** | 学了马上用 | 用一次比看十遍都有效 |
| **建立知识网络** | 新知识和旧知识联系起来 | 有联系更容易记住 |

---

## 一个简单的学习流程

```
1. 学习新知识
     ↓
2. 用自己的话总结
     ↓
3. 找机会应用
     ↓
4. 定期复习
```

**这四步，比单纯地看、听，效果要好得多。**
