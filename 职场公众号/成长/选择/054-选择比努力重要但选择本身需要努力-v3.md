# 选择比努力重要，但选择本身需要努力，我终于明白了

很多人听过这句话：选择比努力重要。

于是他们想：那我就不用努力了，只要选择对了就行。

这是一个误解。

选择确实比努力重要。

但做出好的选择，本身就是一件需要努力的事。

努力什么？

第一，努力收集信息。

好的选择，建立在充分的信息之上。

不了解情况，就不可能做出好的判断。

这需要你花时间去调研、去咨询、去了解。

这不是躺在沙发上想想就能完成的。

第二，努力提升认知。

选择的能力，本质是认知的能力。

认知越高，选择越准。

提升认知，需要学习、思考、复盘。

这同样需要努力。

第三，努力积累实力。

有些选择，是留给有实力的人的。

你没有实力，即使机会来了，也抓不住。

积累实力，需要日复一日的努力。

第四，努力克服恐惧。

很多人知道什么是对的，但不敢选。

因为害怕风险，害怕失败，害怕不确定性。

克服恐惧，需要勇气，也需要底气。

底气来自实力，实力来自努力。

所以，选择比努力重要，但做出好的选择，需要先努力。

不是努力一下就够了，是持续努力，持续积累。

当你的认知、信息、实力都到位了，好的选择自然会出现。

当机会来了，你才有能力抓住。

选择和努力，不是二选一。

是先努力，再选择。

努力是基础，选择是放大器。

没有努力，选择也发挥不了作用。

没有选择，努力也事倍功半。

两手都要，两手都要硬。
